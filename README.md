# 项目简介

## 核心功能
- 基于YOLOv8的木瓜成熟度检测，支持['belum_matang', 'matang', 'setengah_matang']三类分类
- 支持4300张标注数据集
- 提供完整源码、训练好的模型及训练结果
- 包含项目运行教程
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供UI界面，实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有结果
- 可自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 支持选择检测模型
- 支持检测结果保存

## 适用场景
- 木瓜成熟度的自动化检测
- 农业产品品质分级
- 果实成熟度研究
- 计算机视觉教学实践
- 基于深度学习的目标检测应用开发

## 技术特点
- 基于YOLOv8深度学习框架
- 使用Python3.9开发环境
- 集成opencv-python、PyQt5、ultralytics、torch1.9等库
- 提供图形用户界面，操作简便
- 支持多种输入格式和实时检测功能
- 模型训练完成，可直接部署使用
